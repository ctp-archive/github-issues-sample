General: add support for antibody (IgG/IgM) testing results

Issue number 64

lokkju opened this issue on March 27, 2020 at 4:21 am

Labels Org stale

Open in Github

Sort of a meta-issue, but antibody testing results should start being available before too long, and I’d like to make sure the project is ready to track those results (assuming it wants to).

Specifically, adding IgG and IgM tested/positive/negative for upcoming rapid tests would be hugely useful as soon as that data starts becoming available.

If the project is already ready to handle it, it also gives an incentive to anyone who does have that data to make it available to this project, so as to get it in as many hands as possible.

While no national labs seem to be preparing to do antibody testing, at least some independent medical facilities and state labs reported already are, or will soon be.

As an example, this facility expects to have 12K test kits this week: https://www.ocregister.com/2020/03/25/garden-grove-doctor-one-of-first-in-u-s-to-offer-coronavirus-rapid-blood-test/
