There are 3 county sites available: CSBS, CDS, NYT

Use these 3 to create a current day validation check that sum(counties.positives) ~= state.positives and sum(counties.deaths) ~= state.deaths

Based on https://github.com/COVID19Tracking/associated-data/blob/master/county_data/County Data Comparison.ipynb

Uses CSBS, CDS, and NYT data to get a [min, max] range of values for positives, deaths, recovered. If the state #s don’t fall within the summed county aggregate ranges, then an error is triggered (warning for recovered since that data is more sparse).

This test is intentionally imprecise because each sheet is updated at different times. If we can find a better way to make the range of possible values tighter, that would be better.
