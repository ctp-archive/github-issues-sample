Related to this Arizona issue: #755 COVID19tracking’s data quality grade for Arizona is wrong. Arizona is graded as A+, but the state does not report all tests and does not report all negatives. The state reports all positives cases, and a subset of negative results. The reported number for total tests is in fact all positive cases + subset of negatives. This causes problems with positivity calculations for Arizona based off of COVID19tracking data. We need to be accurate when describing the data we aggregate, and stop saying Arizona is reporting all tests when they are not.
