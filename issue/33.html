Just jumping in to say that I looked at our historical data and from the 22nd onward, we have the same numbers.

The shifts forward are because we record a daily number at the same time for all the states (4pm ET): so for us, it didn’t show up in our daily update until the following “business” day. That make sense? It’s annoying, we agree. But it’s how the data structure was originally set up and it would be very difficult to change now.

https://covidtracking.com/data/state/hawaii/
