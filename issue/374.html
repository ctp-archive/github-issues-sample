Thank you for flagging this. It appears that on 5/6, the negative count decreased from 157,030 to 154,686. As a result, we carried the negative value from 5/5, as we do not record data that results in a decrease in negatives over time. This makes it appear as if all new tests are all positive on 5/6.

We reverted to using the NJ data on 5/7 when NJ reported an increase in negatives to 159,023.
