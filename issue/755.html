Arizona:

Describe the problem COVID19Tracking describes Arizona data as including all tests. This is wrong. This error in how COVID19Tracking describes these data has resulted in researchers and analysts who use these data incorrectly calculating test positivity by using all positives, but only a portion of negatives. This issue has not received attention for 15 days. It is an important, but thankfully simple fix. Update the data quality grades.

COVID19Tracking Data quality grades for Arizona have a check for “Machine readable” and “Reporting total test conducted”. There are two issues here:

  • Reporting total test conducted The Director of the Arizona Department of Health Services, Dr. Cara Christ, is quoted in local media today as saying they do not report total tests. The dashboard reports a figure for “total tests”, but Dr. Christ explains in her interview that this is not “total tests”. Some labs are not reporting negative results to the state, so DHS is unable to include those negative results in the total. They report all positive tests, but only some of the negative tests, making the “total test” figure the sum of the all the tests they have, rather than the actual total conducted. Please update the data quality grade accordingly.

  • Machine readable Confirmed/probable cases and confirmed/probable deaths are reported as a tableau vizintooltip. The data itself is an image of numbers, with no alt text. I’m unable to parse the image with OCR, so I’m not sure how this is machine readable. I could be missing something, though.

Link to data source

  • Total tests See azfamily.com

Dr. Christ says some labs across the state report all their negative and positive cases while other labs only report their positive cases.

When calculating the positivity, the state does not include positive cases unless the lab also provides negative cases. Johns Hopkins uses all data collected.

Dr. Christ says using the positives without the negatives creates a falsely high number.

“You could have one person get tested, and it gets reported to the department, but there could have been 1,000 people tested. If those negatives don’t get included, you’d have 100% positivity, and that’s not accurate,” says Dr. Christ.

Dr. Christ says more in the video. The relevant section starts at 4:06. Dr. Christ is saying that you (the source of JHU’s calculation) do not, in fact, have the denominator for the percent positivity statistic (total tests) because they (DHS) are not reporting it. This should be reflected in the data quality grade.
