Hi all, I run covid-19-tools which is more of a python library for working with covid-data in object oriented structures. It does a good job of processing JHU’s data as well. One of the things we can do quickly is run the state-by-state comparison, just after midnight EST.

In the following, the first number is JHU’s state death count (summed up from the county level). The second number is the death count from Covid19Tracking’s JSON API. Third number is the delta. As you can see there are some very good perfect matches here, but NY is obscenely off (JHU problem).

I’m willing to look at calibrating the models a little bit. In fact, calibrating the JHU county level data to the Covid19Tracking because if we’re interested in the best possible data, we ought to be synthesizing as many sources as possible.
