First, thank you for putting together the covid tracking resources. I think this is a great resource and a tremendous undertaking.

I believe, however, that there is an issue with historical data. States seem to be updating their older case counts as new information becomes available, but the state history series that you release do not update so regularly. For example, the data at the state daily API for New York on 3/12 reports 309 total tests (216 positive, 92 negative). However, the data from New York state (https://coronavirus.health.ny.gov/county-county-breakdown-positive-cases) show 2836 tests that day. The most recent numners do match, so I believe the problem is updates to the prior test numbers, as more information gets reported to the state.

It may not be possible to fix this problem by updating all the historical data, but perhaps you could confirm that this is an issue, and that I should trust the state reports when they are available. Is that right?
